RSA学习

2017-10-10密码学

第一步：生成公钥

首先选择两个大素数p和q
p = 53   q = 59
生成公钥：pq = n （53*59 = 3127）
选取一个小指数e（e必须满足以下几点）:
必须是整数
不能是n的因子
1 < e < Ф(n)
这里选3

第二步：生成私钥

Ф(n) = (p-1)(q-1)
Ф(n) = (53-1)(59-1) = 3016
计算私钥d
d = (2Ф(n)+1)/2=(2*3016+1)/3 = 2011

屡一下思路

公钥组成
n  e
私钥组成
D

第三步：加解密数据

加密数据
c(密文) = 89e mod n = 893 mod 3127 （89是明文）
解密数据
明文 = cd mod n = 13942011 mod 3127 = 89
